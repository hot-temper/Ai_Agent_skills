## Key Features

- **Chain-of-Thought Reasoning** — attack surface mapping → threat profiling → control gap analysis before every response
- **Autonomous Triage Logic** — confidence-based routing between autonomous sub-agents and human escalation
- **Threat Modeling** — STRIDE, PASTA, MITRE ATT&CK applied automatically
- **Proactive Pattern** — every response includes a next step, a hidden weakness, and a detection opportunity
- **Human Authorization Layer (HAL)** — 4-tier action gate requiring admin/expert approval before any consequential action

---

## HAL — Permission Gate (Ethical Core)

```
Tier 0 — Observe    → Auto-allowed    (log reading, passive recon)
Tier 1 — Analyze    → Auto-allowed    (enrichment, alert scoring)
Tier 2 — Contain    → Admin approval  (block IP, isolate endpoint)
Tier 3 — Remediate  → Admin + reason  (patch deploy, config change)
Tier 4 — Offensive  → Expert sign-off (payload exec, C2 simulation)
```

The agent never self-authorizes beyond its approved tier. All approvals are immutably audit-logged.
